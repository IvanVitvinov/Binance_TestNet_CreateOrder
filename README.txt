Структура:
    1. В документе имеется в data храниться файл config.py для API ключей
    2. binance_TestNet.py - код написанный для тестового баланса
    3. real_order.py - код для реального счета Бинанс. (Не успел переписать на библиотеку binance-python)
    4. test_for_real_order.py - несколько тестов к скрипту для реального бинанса.

Использование:
    1. В папке data заменить API ключи на свои. (Для тест нета оставил действующие ключи)
    2. Открыть скрипт binance_TestNet, в нем можно изменить параметры в функции main(), и посмотреть на отработку ошибок. 
    3. real_order тот же самый скрипт, только для реального счета бинанс. Хотел переписать его на библиотеку binance-python, но в задании стоят сроки до 28 мая. Не успел
    4. test_for_real_order.py - Скрипт с 3 простыми тестами для real_order. При наличии времени можно увеличить объем данного скрипта.
